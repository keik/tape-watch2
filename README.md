[tape](https://github.com/substack/tape) with watch mode.


## Installation

```
npm install tape-watch2
```


## Usage

Run `tape-watch` command instead of `tape` command like:

```
tape-watch 'test/test-*.js'
```

For more details, run `tape-watch --help`
